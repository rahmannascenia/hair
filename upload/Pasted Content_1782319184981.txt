# 🏭 Hairlan International ERP — New Functionality Ideas for Complete Visibility

## Category 1: 🔍 LOT-LEVEL GRANULAR TRACKING (The Backbone)

### 1.1 Lot Journey Tracker (Single-Lot 360° View)
**What it does:** Enter a lot number → see a visual **timeline/pipeline** showing where the lot is RIGHT NOW, with every transaction it went through.

**Data shown per lot:**
- **Current stage** in the 7-state lifecycle (Created → Washing → Distributed → In-Factory → Half-Finish Returned → In Final Production → Finished)
- **Total days elapsed** since creation (flag if >45 days — KPI target)
- **Weight at each stage:** Raw kg → Washed kg → Distributed kg → Returned kg → Finished kg (with % loss at each stage)
- **Cumulative wastage** across all stages (kg and %)
- **Current custodian** — who has this lot right now? (Supplier → Warehouse → Washing Plant → PM → Head Leader → Line Leader → Supervisor → Worker → Back to QC → Dhaka Factory)
- **Valuation at current stage** — landed cost + stage uplifts (Raw=0, Washed=+200, Distributed=+350, In-Factory=+600, Half-Finish=+1500, Final=+4500 BDT/kg)
- **Realisable value** — if sized, Σ(size_kg × size_rate) across all size categories

**Visual:** A horizontal stepper/pipeline with dots for each stage, colored green (completed), gold (current), gray (pending). Clicking a stage shows the detailed records at that stage.

### 1.2 Lot Traceability Drill-Down
**What it does:** From any lot, drill into every person who touched it.

- **Which washing operator** washed this lot? (WashLog.operator, wastage %, date)
- **Which Head Leader** received it? (Phase1Distribution where fromRole="PM", toName)
- **Which Line Leaders** distributed it? (Phase1Distribution where fromRole="Head Leader")
- **Which factories/supervisors** processed it? (FactoryDailyRecord filtered by lotId → shows factory name, supervisor, dates)
- **Which workers** processed it? (WorkerDailyEntry → shows worker name, date, A/B/C grades, output, wastage, pay)
- **Phase 2 sizing results** (Phase2Job → size distribution, margin)
- **Sales contracts** fulfilled from this lot (Sale linked to lot)

**Impact:** "Enter LOT-20260412-01 → see that 47 workers across 5 factories processed it, with 62% A-grade, 12.3% total wastage, and it yielded 180 kg of finished goods worth ৳22.5 lakh."

### 1.3 Lot Balance Reconciliation
**What it does:** Cross-check that input = output + wastage at EVERY transfer point for a lot.

- Procurement raw weight vs. Wash input weight
- Wash input vs. Wash output (washing loss)
- Wash output vs. Total distributed to Head Leaders
- Distributed to factories vs. Returned by factories
- Returned vs. Phase 2 input
- Phase 2 input vs. Total sized + combing loss
- **Alert:** Any discrepancy flagged as "INVESTIGATE" with kg variance

### 1.4 Lot Quality Inheritance Analysis
**What it does:** Flag lots where C-grade concentration is >50% (per business rule R18 — lot quality affects all workers).

- Show A/B/C % per lot across all workers/factories
- If C% > 50%, flag: *"This lot's raw material quality may be causing low grades — workers may not be at fault"*
- Compare grade distribution across lots from the same supplier/country

## Category 2: 👷 WORKER-LEVEL VISIBILITY

### 2.1 Worker Profile Card (Worker ID → 360° View)
**What it does:** Enter a worker ID → see everything about that worker.

**Data shown:**
- **Basic info:** Name, Factory, Supervisor, Line Leader, Group Head, bKash number, Joining date, Active status
- **Current month's performance:** Days present, Grade distribution (A/B/C days), Total output (A+B+C kg), Total pay, Attendance bonus eligibility, Performance bonus eligibility
- **Quality trend:** Last 30 days — daily A/B/C grades as a color-coded bar (green=A, gold=B, red=C) — instantly see patterns
- **Average daily output** (grams) — compare against factory average
- **Earnings trend:** Monthly earnings over last 6 months (bar chart)
- **Attendance streak:** Current consecutive days present / 30-day target for bonus
- **Auto-C grade count:** How many times output <50g triggered automatic C grade
- **Lots worked on:** List of all lot numbers this worker processed, with date, input kg, output grades

**Impact:** "Worker W-003 → works at F-01 Fatema, 28 days present, 65% A-grade, avg 220g/day, earned ৳2,450 this month, eligible for attendance bonus, processed LOT-20260412-01 and LOT-20260503-03."

### 2.2 Worker Performance Leaderboard
**What it does:** Rank workers within a factory, across a Line Leader's territory, or company-wide.

- Sortable columns: A-grade %, Total output (kg), Total pay, Attendance rate, Consistency score
- Filter by: Factory, Line Leader, Group Head, Date range, Grade threshold
- Highlight top 10 and bottom 10 performers
- Show which workers are at risk of termination (continuous C-grade, per business rule)

### 2.3 Worker Grade Dispute Tracker
**What it does:** Since QC accuracy is ~70%, allow workers/supervisors to flag disputed grades.

- **Flag button** on any daily entry: "Dispute Grade"
- **Reason field:** "Lot was poor quality", "Weight was correct", "Was sick that day", etc.
- **Review queue** for Head Leader / QC manager to review disputes
- **Resolution:** Upheld / Overturned with updated grade
- **Dispute rate per QC inspector** — if one QC has high overturn rate, flag for retraining

### 2.4 Worker Attendance Calendar (Heatmap)
**What it does:** Visual calendar for each worker showing present (green), absent (red), holiday (gray).

- Monthly view for the last 12 months
- Hover over any day → see that day's grade, output, pay
- Running attendance streak counter
- "At risk" workers highlighted — those who will lose attendance bonus if absent 1 more day

## Category 3: 🏠 FACTORY/SUPERVISOR-LEVEL VISIBILITY

### 3.1 Factory Daily Scorecard
**What it does:** A single-day view for each factory showing everything that happened today.

- **Summary tiles:** Total workers present, Total input given, Total graded output (A/B/C), Wastage %, Grand total payroll, A-grade % (with threshold indicator)
- **Per-worker table:** Each worker's input, A/B/C output, balance check, pay
- **Supervisor pay calculation:** Hosting allowance + Performance bonus (with eligibility criteria shown)
- **WIP status:** Material received vs. completed vs. remaining (progress bar)
- **Date navigation:** ← Previous day | Today | Next day →
- **Comparison vs. yesterday:** "Output up 12%", "Wastage down 3%"

### 3.2 Factory Performance Dashboard (30-Day Rolling)
**What it does:** Rolling 30-day view per factory.

- Daily output trend (line chart)
- A/B/C grade % trend (stacked area chart)
- Wastage trend (bar chart with tolerance line at 15%)
- Cost/kg trend vs. target (320 BDT/kg)
- Worker attendance trend
- "At risk" alerts: A-grade % dropping below 60%, cost/kg above target, worker turnover

### 3.3 Supervisor P&L Statement
**What it does:** Per supervisor, show a mini profit/loss — input material value vs. output graded value vs. labor cost.

- Input material cost = input_kg × landed_cost_per_kg
- Output value by grade = A_kg × rate_A + B_kg × rate_B + C_kg × rate_C
- Labor cost = total worker wages + supervisor allowance + fuel + transport
- **Net margin** = Output value - Input cost - Labor cost
- Per the business document: "Supervisors generating profit receive production bonus; those generating consistent loss may be terminated"

### 3.4 Multi-Factory Comparison View
**What it does:** Side-by-side comparison of all factories on key metrics.

- Table: Factory | Workers | Input | A% | B% | C% | Wastage% | Cost/kg | Output | Payroll | Margin
- Sortable by any column
- Best/worst factory highlighted per metric
- Filter by: Line Leader, Group Head, Date range

## Category 6: 🔐 ROLE-WISE INTERFACE & AUTHENTICATION

### 6.1 Role-Based Login System
**Roles** (from the organizational hierarchy):
- **Owner / Managing Director** — Full access to everything + approval workflows + command center
- **Production Manager (PM)** — Distribution, washing, lot tracking, factory oversight (NOT salary approval, NOT financial reports)
- **Group Head (Head Leader)** — Only their territory's factories, LLs, workers, daily records (read-only on pricing/financials)
- **Line Leader** — Only their factories' daily records, worker lists, WIP tracking (no pricing, no financials)
- **Factory Supervisor** — Only their factory's daily records and worker list (can enter daily production data)
- **QC Inspector** — Can view all factories (for grading), enter grades, can't see payroll or financials
- **Accountant** — Payroll, procurement, sales, FX, financial reports (can't modify production data)
- **Admin / HR** — Worker management, factory management, org structure, attendance

### 6.2 Role-Specific Home Pages
Each role sees a **different default dashboard** when they log in:

- **Owner:** Command Center (Category 4.3) — everything at a glance
- **PM:** Today's production plan, distribution status, washing queue, factory alerts
- **Head Leader:** Territory dashboard, today's collection summary, LL performance, approval queue for daily records
- **Line Leader:** Today's route (factories to visit), material to distribute, collections summary, WIP by factory
- **Supervisor:** Today's worker list, input received, enter worker output, return summary
- **QC:** Grade entry queue (list of factories whose workers need grading today), grade distribution stats
- **Accountant:** Pending salary sheets, FX position, monthly financial close checklist

### 6.3 Approval Workflow Engine
- **Salary sheet approval:** Supervisor submits → Head Leader reviews → PM reviews → Owner approves → bKash disbursement
- **Purchase approval:** New supplier/PO → PM recommends → Owner approves
- **Exception approval:** Shortage/reject > threshold → flagged → PM/Owner approves write-off
- **Lot status transitions:** Certain transitions (e.g., lot to "Finished") require PM approval

### 6.4 Audit Trail
- **Every data change** logged: who changed what, when, old value, new value
- **Viewable per record:** "Show history" button on any lot, daily record, sale, etc.
- **Role-based:** Only Owner/Accountant can see full audit trail; others see their own changes only

## Category 8: 🎯 QUALITY CONTROL ENHANCEMENTS

### 8.1 Grade Distribution Analytics
- **Per lot:** A/B/C % with histogram
- **Per factory:** Rolling 30-day A/B/C % trend
- **Per Line Leader:** A/B/C % across all factories
- **Per QC inspector:** Grade distribution — are some QCs stricter than others?
- **Per supplier/country:** Grade % of lots from each source (some suppliers consistently produce better raw material)

### 8.2 Grade Quality Scoring (Beyond A/B/C)
- **Within A-grade:** A+, A, A- based on consistency (if always A, they're A+)
- **Worker quality score:** Composite score = (A% × 3 + B% × 2 + C% × 1) / total days
- **Factory quality score:** Weighted average of all workers' scores
- **Trend indicator:** ↑ Improving, → Stable, ↓ Declining

### 8.3 Reject & Shortage Investigation Module
- **Separate tracking** for Reject (quality failure) vs. Shortage (unaccounted loss) — currently mixed
- **Reject recovery tracking:** Can rejected material be re-processed? What's the recovery rate?
- **Shortage hotspots:** Which factories/LLs/supervisors have the highest shortage rates?
- **Investigation workflow:** Flag → Assign investigator → Findings → Resolution → Close
- **Theft risk scoring:** High shortage + low QC grade + high turnover = elevated theft risk

### 8.4 QC Accuracy Tracking
- **Re-grading system:** Random sample of 10% of workers re-graded by a second QC
- **Discrepancy rate per QC:** If QC-1 grades A but QC-2 grades B, that's a discrepancy
- **QC training needs:** QCs with >30% discrepancy rate flagged for retraining
